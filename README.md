## Descrição do projeto 

<p align="justify">
  Há dois anos atrás, criei uma aplicação mobile para Android, usando Java. O aplicativo era um gerenciador de time de futsal. Com ele você podia cadastrar seus jogadores e algumas informações básicas que eram: nome, posição em que jogavam, pé dominante e até uma foto dele. Se não me engano, acho que cheguei a terminar, na época que fiz, estava aprendendo coisas até avançadas eu diria para aplicação mobile como usar o banco de dados local (SQLite) e manipular funções do celular como o uso de câmara. 
  
  Desta vez, foi reproduzir o que tinha feito anteriormente mas dessa vez, irei utilizar o React Native com TypeScript. Vai ser um projeto de fim de semana. 
</p>
